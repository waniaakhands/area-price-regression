# area-price-regression
A Machine Learning project that uses Linear Regression to predict house prices based on property area. Includes model training, prediction, evaluation metrics (MAE and MSE), and data visualization using Matplotlib.

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Skills Demonstrated

- Machine Learning
- Linear Regression
- Data Visualization
- Model Evaluation (MAE, MSE, R² Score)
- Data Preprocessing
- Predictive Modeling
